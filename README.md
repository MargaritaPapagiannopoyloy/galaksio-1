#Galaksio
> An easy to use GUI for running Galaxy workflows.

Galaksio is a web application that simplifies the usage of the Galaxy bioinformatics platform (https://usegalaxy.org/).

Galaksio has been developed as part of the B3Africa Project (http://www.b3africa.org/), which has received funding from the European Unionâ€™s Horizon 2020 research and innovation programme.
